# Swift LeetCode Interview Prep

A comprehensive Swift workbook for software engineering interview preparation, featuring high-frequency LeetCode problems organized as individual Xcode Playground files. Perfect for FAANG/MAANG interview prep.

## 🎯 Purpose

This repository serves as a personal workbook for mastering algorithmic problem-solving in Swift. Each LeetCode problem is implemented in its own standalone Xcode Playground, allowing for immediate execution, testing, and experimentation.

## 🚀 How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/Swift-LeetCode-Interview-Prep.git
   cd Swift-LeetCode-Interview-Prep
   ```

2. **Open any playground in Xcode:**
   - Double-click any `.playground` file to open it in Xcode
   - Run the code immediately with ⌘+Shift+Return
   - Modify and experiment with the solutions

3. **Navigate problems by difficulty:**
   - **Easy/**: Fundamental problems for building confidence
   - **Medium/**: Core interview problems
   - **Hard/**: Advanced problems for senior positions

## 🔄 Adding New Problems

To add a new problem:
1. Create a new playground: **File > New > Playground** in Xcode
2. Name it following the pattern: `LC[XXX]_[ProblemTitle].playground`
3. Move it to the appropriate difficulty folder
4. Follow the existing template structure

## 🎓 Study Tips

- Start with Easy problems to build confidence
- Focus on understanding patterns rather than memorizing solutions
- Practice explaining your approach out loud
- Time yourself to simulate interview conditions
- Review and optimize solutions for better time/space complexity
